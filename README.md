# Getting Started

##### For running locally

Run using `mvnw spring-boot:run`

Alternatively, you can build the JAR file with 
`mvnw clean package` and then run the JAR file, as follows:
`java -jar target/datemath-api-0.0.1-SNAPSHOT.jar`
